

*   which $SHELL : Prints the name of the shell which we are using. 

*   echo         : The echo command is used to display a line of text that is passed in as an argument.   
                    
                    echo hello_world 

*   nano          : GNU nano is an easy to use command line text editor for Unix and Linux operating systems.

                    nano himom.sh   

*   #!            : shebang is used to tell the kernel which interpreter should be used to run the commands present in the file.

*   #!/bin/bash    : when used in scripts is used to instruct the operating system to use bash as a command interpreter.

*   ctrl + X        : Used to save.
                    
                    To run the script, type bash <script_name.sh>

*   sleep           : Sleep is a very versatile command with a very simple syntax. It is as easy as typing sleep N . This will pause your script for N seconds, with N being either a positive integer or a floating point number. 

*   ls -l           : summarizes the most important information about the file on a single line.

*   chmod           : It is an access control command used to change the access permissions of files and directories.
                    
                     For eg: chmod +x <filename.sh> {This will change the read & write only permission of a file to read, write and executable}


*   clear           : Used to clear the screen.

*   pwd             : print working directory

*   ls              : list the contents of your current working directory

*   cd              : change directory

*   cd ..           : to go to the previous directory

*   whoami          : Displays who we are logged in as.

*   date            : Displays the date.

*   cp              : stands for copy.

*   cat             : Cat(concatenate) command is very frequently used in Linux. It reads data from the file and gives their content as output. It helps us to create, view, concatenate files.

*   rm              : stands for remove.

*   ctrl + c        : stops the command being run on BASH.

* man <command_name>: to get help with a particular command, it launches the manual for it.

*<command_name> --help : to get help with the particular command, launches a brief version. 

* appropos          : appropos and type key words related to what you want to do. It will display relatable commands.
